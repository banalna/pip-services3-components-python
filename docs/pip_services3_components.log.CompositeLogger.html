class pip_services3_components.log.CompositeLogger.CompositeLogger(references=None)¶

Bases: pip_services3_components.log.Logger.Logger, pip_services3_commons.refer.IReferenceable.IReferenceable

Aggregates all loggers from component references under a single component.

It allows to log messages and conveniently send them to multiple destinations.

### References ###
  • :logger::*:1.0 (optional) ILogger components to pass log messages
Example:
class MyComponent(IConfigurable, IReferenceable):

_logger = CompositeLogger()

def configure(self, config):
self._logger.configure(config)
def set_references(self, references):
self._logger.set_references(references)
def my_method(self, correlation_id):
self._logger.debug(correlationId, “Called method mycomponent.mymethod”)
