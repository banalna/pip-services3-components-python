class pip_services_components.config.JsonConfigReader.JsonConfigReader(path)¶

Bases: pip_services_components.config.FileConfigReader.FileConfigReader

Config reader that reads configuration from JSON file.

The reader supports parameterization using Handlebar template engine.

### Configuration parameters ###

  • path: path to configuration file
  • parameters: this entire section is used as template parameters
  • …
Example:

======== config.json ====== { “key1”: “{{KEY1_VALUE}}”, “key2”: “{{KEY2_VALUE}}” } ===========================

configReader = JsonConfigReader(“config.json”) parameters = ConfigParams.from_tuples(“KEY1_VALUE”, 123, “KEY2_VALUE”, “ABC”) configReader.read_config(“123”, parameters)
